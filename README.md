## Getting started

```bash
git clone https://github.com/luisMakesIt/react-crud-tailwind-localstorage-context-api.git
cd react-crud-tailwind-localstorage-context-api
```

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```
